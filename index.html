Latar Belakang

Pada tahun 2017 , bahkan anak-anak pun sudah terbiasa menggunakan perangkat telepon pintar/smartphone, sebagian besar dari smartphone tersebut dijalankan dengan sistem operasi Android atau IOS. Dalam smartphone terdapat aplikasi-aplikasi yang dapat digunakan untuk berbagai keperluan. Salah satu jenis dari aplikasi-aplikasi tersebut adalah game. Selain sebagai perangkat untuk membantu pekerjaan, smartphone juga dapat digunakan sebagai perangkat hiburan dengan adanya game. Game-game ini sama hanya dengan aplikasi-aplikasi jenis lain, dapat dengan mudah didownload dari Google Play. Selain sebagai wahana permainan dan hiburan,game juga dapat dijadikan peluang bisnis dan iklan. Penulis tertarik untuk membuat sebuah game khusus smartphone dengan sistem oprasi android karena adanya peluang tersebut. Namun untuk membuat game tidaklah mudah. Walaupun dengan adanya progam game engine gratis seperti unity, ada tantangan-tantangan yang akan dialami oleh seorang pembuat game atau game developer.jika pembuat game tidak mampu melewati tantangan-tantangan ini,maka game tidak akan selesai. Salah satu tantangan tersebut adalah penyimpanan informasi pemain. Sebagian besar game tidak bisa diselesaikan pada waktu singkat. Pemain juga mempunyai kesibukan masing-masing yang berarti waktu mereka bermain terbatas. Tentunya ketika pemain ingin bermain,mereka ingin melanjutkan game yang terakhir dimainkan.maka dari itu sistem save and load . Save menuliskan data-data terakhir pemain kedalam sebuah file.sementara load berfungsi umtuk membaca dan memproses data-data tersebut kedalam game.

    Landasan Teori

    Pengertian Game

    Game berasal dari bahasa Inggris. Dalam kamus bahasa Indonesia istilah Game berarti permainan. Menurut Suryawan, dan Jalaluddin(2013: 489), permainan sebuah sistem dimana pemain terlibat dalam konflik buatan. Pemain berinteraksi dengan sebuah sistem dan konflik dalam permainan. Dalam permainan terdapat peraturan yang bertujuan untuk membatasi perilaku pemain dan menentukan permainan.

    Pengertian Android

    Android merupakan generasi baru platform mobile berbasis Linux yang mencakup sistem operasi, middleware, dan aplikasi (Nazrudin Safaat, 2012), Menurut Damarullah,dkk (2013), Android awalnya dikembangkan oleh Android Inc. dengan dukungan finansial dari Google yang kemudian membelinya pada tahun 2005. Sistem operasi ini dirilis secara resmi pada tahun 2007, bersamaan dengan didirikannya Open Handset Alliance, konsorsium dari perusahaan-perusahaan perangkat keras, perangkat lunak, dan telekomunikasi yang bertujuan untuk memajukan standar terbuka perangkat seluler.

    Pengertian 2D

    Game 2D adalah game yang secara matematis melibatkan 2 elemen koordinat kartesius yaitu x dan y,sehingga konsep kamera pada game 2D hanya menentukan gambar pada game yang dapat dilihat oleh pemain.

    Pengertian Unity

    Unity merupakan sebuah software engine untuk mengembangkan game dengan genre casual, AR (Augmented Reality) dan VR (Virtual Reality). Hasil game dari Unity 3D ini bersifat cross-platform. Yang berarti Anda bisa mempublikasikan game ke beberapa platform. Contohnya seperti:Windows, Android, Mac, PS3 dan lainnya. Tidak hanya membuat desain grafis untuk game, Anda juga bisa menambahkan script untuk memprogamnya. Bahkan dari sisi pengaturan sudah disediakan beberapa tools bawaan.

    Bab 3

    Manajemen Produksi

    1.Kerangka Kerja

    Berikut akan dijelaskan kerangka kerja dalam pembuatan game ini. Kerangka akan dibagi dalam bentuk langkah-langkah, kemudian setiap langkah tersebut akan dijelaskan.

    2.Perancangan Cerita

    Akan lebih mudah untuk membuat sebuah game jika pembuat sudah merancang tentang apakah game tersebut. Tidak semua game harus bercerita, namun untuk game yang akan dibuat dalam penelitian ini, ada cerita yang terkandung

    3.Perancangan Gameplay

    Sebelum game dibuat, harus diketahui seperti apa gameplaynya. Gameplay adalah cara sebuah game berjalan atau cara memainkan sebuah game. Apakah game akan berupa game aksi, atau game balap dan sebagainya

    4.Perancangan dan pembuatan karakter

    Setelah mendapatkan cerita dan gameplay maka dapat dilanjutkan dengan merancang dan membuat karakter yang sesuai. Karakter-karakter yang dibuat adalah karakter pemain dan karakter yang dijalankan otomatis oleh komputer. Kemudian karakter dibuat sesuai dengan rancangan menjadi model 2D.

    5.Membuat model-model tambahan

    Karakter yang telah dibuat tidak dapat berbuat apa-apa di dalam game jika tidak ada benda lain seperti lingkungannya. Maka setelah mendapatkan model karakter yang sesuai, langkah berikutnya adalah membuat model-model lain seperti model lingkungan, model garis finish, gedung, lintasan, dan sebagainya

    6.Penyusunan asset

    Karakter, model, antarmuka dan benda-benda lain yang sudah selesai dan akan digunakan di dalam game diistilahkan sebagai asset. Asset-asset tersebut kemudian diimport ke dalam projek game dalam unity dan disusun rapi sedemikian rupa

    7.Penulisan script

    Asset-asset yang telah dibuat tidak dapat berbuat apapun jika tidak diberikan perintah. Perintah tersebut diberikan dengan script. Script-script ini dibuat dan ditempelkan ke asset-asset yang sesuai sehingga game dapat berjalan

    8.Uji coba dan perbaikan

    Sangat sulit untuk membuat game yang sempurna dalam sekali coba. Game yang selesai dibuat kemudian diuji coba dengan cara dimainkan di media yang dituju. Untuk game ini media tersebut adalah perangkat android. Dari uji cona tersebut dapat ditemukan kesalahan-kesalahan baik dalam script, model, animasi, dan sebagainya. Kesalahan-kesalahan tersebut kemudian diperbaiki sebelum diuji coba lagi. Proses ini diulangi terus menerus hingga semua kesalahan yang ditemukan telah diselesaikan.

    9.Evaluasi

    Selama membuat game, pembuat akan mendapatkan pengalaman. Dari pengalaman tersebut kemudian dapat ditarik pelajaran-pelajaran yang dapat diterapkan untuk meningkatkan kualitas game-game lain yang akan dibuat kedepannya

    Analisis SWOT

    Sebagai acuan untuk menghadapi persaingan dalam bidang usaha, setiap untuk memulai usaha, kami harus melakukan analisis terhadap lingkungan atau persaingan melalui SWOT

    • Strengths

      a) Dapat mendukung hobi pelajar atau mahasiswa yang hobi bermain game

      b) Dapat mengembangkan kreatifitas untuk membuat game yang bermanfaat

    • Weakness

      a) Masih ada pembatas antara jadwal belajar dengan jadwal bermain game

      b) Peminat game offline masih sedikit dibanding peminat game online

    • Opportunities

      a) Di zaman sekarang industri game sedang berkembang, khususnya game online

      b) Game memiliki kelebihan selain untuk bersenang-senang juga dapat memberi pengetahuan

    • Threats

      a) Banyak saingan dari toko-toko penjual Game dan software

      b) Masih kurangnya pembuat game indie dan developernyat

    Penutup

    Kesimpulan

    Penggunaan Unity sebagai game engine untuk pembuatan game dua dimensi dapat dikatakan cukup efisien. Sebagian besar metode yang diterapkan untuk membuat game tiga dimensi dapat diterapkan juga untuk membuat game dua dimensi. Selain itu, masukan dari perangkat mobile juga dapat terintegrasi dengan baik. Game yang dibuat terbukti dapat mengimplementasikan penggunaan akselerometer sebagai metode masukan untuk meningkatkan pengalaman pengguna

    Saran

    Pengembangan selanjutnya direkomendasikan untuk menyajikan perbedaan resolusi dan orientsi pada game sehingga pengguna dapat memainkan game secara maksimal meskipun menggunakan perangkat yang berbeda. Selain itu implementasi akselerasi juga perlu diterapkan pada game dengan genre lain untuk melihat potensi dari pengembangan game dengan masukann ini
